## 🔧 設定方法

### 手順1: カスタムステートの作成

1. **データベースを開く**
   - SRPG Studioでプロジェクトを開く
   - `データ設定` → `ステート` を選択

2. **新規ステートを作成**
   - 「ステートの作成」ボタンをクリック
   - 名前を設定（例：「[ステート名]」）

3. **カスタムパラメータを設定**
   ```json
   {
       "[パラメータ1]": [値],
       "[パラメータ2]": [値],
       "[パラメータ3]": [値],
       "[パラメータ4]": [値]
   }
   ```

4. **その他の設定**
   - [重要な設定がある場合のみ記載]
     ※[理由説明]
   - アイコンを設定（任意）

### 手順2: カスタムスキルの作成

1. **データベースを開く**
   - `データ設定` → `スキル` を選択

2. **新規スキルを作成**
   - 「スキルの作成」ボタンをクリック
   - 名前を設定（例：「[スキル名]」）

3. **キーワードを設定**
   - キーワード欄に `[プラグインキーワード]` と入力（大文字小文字を正確に）

4. **カスタムパラメータを設定**
   ```json
   {
       "[パラメータ]": [値]
   }
   ```
   ※ `[パラメータ]`の値は手順1で作成したステートのIDに置き換えてください
